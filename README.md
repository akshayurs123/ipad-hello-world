# ipad-hello-world

A minimal SwiftUI "Hello, World!" app for iPad.

## Requirements

- Xcode 15+
- iOS/iPadOS 17.0+
- Swift 5.0+

## Getting Started

1. Open `HelloWorldiPad.xcodeproj` in Xcode
2. Select an iPad simulator or connected iPad device
3. Press `Cmd+R` to build and run

## Project Structure

```
HelloWorldiPad/
├── HelloWorldApp.swift   # App entry point
├── ContentView.swift     # Main UI view
├── Info.plist            # App configuration (iPad-only)
└── Assets.xcassets/      # App icons and assets
```
